# Todoist App by @matisantillan11

- In this repo we will use requirer to interact with de app in the CL

* Packages used: Colors, inquirer, uuid

- When app started, you gonna see a Menu like this:

![image](https://user-images.githubusercontent.com/50279534/112873776-79556380-9098-11eb-81f2-1f8b3690d713.png)

- If you select the "Task List" option and there is no task, you will see a message like this:

![image](https://user-images.githubusercontent.com/50279534/112873980-b7528780-9098-11eb-8e35-7b6c816e4597.png)

- To create a task, select the option and we will ask you for a title for it. 

![image](https://user-images.githubusercontent.com/50279534/112874153-f254bb00-9098-11eb-8dc7-2c2e62c35eda.png)

- To complete a task, choose the option and select from a list of tasks which one you want to complete

![image](https://user-images.githubusercontent.com/50279534/112874291-229c5980-9099-11eb-863c-816ef70f7e14.png)

- Once you complete a task, if you re-list it, you will see it marked as [COMPLETED]

![image](https://user-images.githubusercontent.com/50279534/112874393-48296300-9099-11eb-930f-adf16c2f754e.png)


- Deleting a task is the same procedure as completing a task. You must choose the task to delete from a list of them with the obvious difference that if you want to see it in the task list, the task will not exist

![image](https://user-images.githubusercontent.com/50279534/112874557-773fd480-9099-11eb-87fe-166994fc2852.png)
